# Platform — 平台前端

游戏广场 + 一句话输入 + iframe 游戏展示容器。

## 页面结构

```
┌─────────────────────────────────────┐
│  🏰 GameCastle                      │
│  ┌─────────────────────────────┐    │
│  │  一句话输入框                │    │
│  │  [ 做一个打砖块游戏 ...] [生成] │    │
│  └─────────────────────────────┘    │
│                                     │
│  ┌──────┐ ┌──────┐ ┌──────┐       │
│  │ 游戏1 │ │ 游戏2 │ │ 游戏3 │       │
│  └──────┘ └──────┘ └──────┘       │
│                                     │
│  ┌─────────────────────────────┐    │
│  │  游戏试玩区 (iframe)         │    │
│  │  ┌───────────────────────┐  │    │
│  │  │  GDJS Runtime         │  │    │
│  │  │  + project.json       │  │    │
│  │  └───────────────────────┘  │    │
│  └─────────────────────────────┘    │
│                                     │
│  💬 "再难一点"  [发送]              │
└─────────────────────────────────────┘
```

## 文件

| 文件 | 说明 |
|------|------|
| `public/index.html` | 单页应用，暗色主题（紫黑配色），包含输入区、游戏卡片网格、全屏游戏播放器 |

## 技术选型

MVP 阶段：纯 HTML5 + Vanilla JS（零构建，直接跑）。

## 游戏展示容器

iframe 沙箱隔离加载 game.html，GDJS Runtime 在其中渲染：

```html
<iframe id="gameFrame" src="/output/game.html"></iframe>
```

计划通过 postMessage 通信：
- 平台 → 游戏: 加载 project.json
- 游戏 → 平台: 分数上报、游戏结束事件
